## ğŸš¦ Getting Started

### Prerequisites
- Python 3.8+
- Node.js 16+
- MongoDB 4.4+
- Modern web browser

### Installation
```bash
# Clone repository
git clone https://github.com/your-username/qofa.git
cd qofa

# Backend setup
cd backend
pip install -r requirements.txt
python server.py

# Frontend setup
cd ../frontend
yarn install
yarn start
```
